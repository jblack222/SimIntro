######### SIMINTRO README ##############
# This code is for simulating panmixia in SNP data.
# This code includes a programs script (sims1_prog.R), and a script for adapting and processing of SNP data (SimIntro.R)
# SimIntro.R published in: Black, J. G., Cooper, S. J. B., Schmidt, T. L., Weeks, A. R., in review (2024). Assessing genetic mixing in divergent and highly structured populations of the threatened southern brown bandicoot (Isoodon obesulus). Evolutionary Applications, 00.
# sims1_prog.R published in: Weeks, A.R., Stoklosa, J., Hoffmann, A.A., (2016). Conservation of genetic uniqueness of populations may increase extinction likelihood of endangered species: the case of Australian mammals. Frontiers in Zoology 13, 31. https://doi.org/10.1186/s12983-016-0163-z
